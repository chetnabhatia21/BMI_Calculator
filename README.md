# BMI Calculator (python)
A simple python program to calculate the Body Mass Index (BMI) of a person and then categorize users calculated BMI into heath range 
## features 
-Accepts users input for weight and height in kilogram and meter respectively 
-Calculates the BMI accurately using the standard formula 
-Displays BMI category based on WHO classification 

## Technologies Used 
-python3.13.5
-Basic conditions, input/output  and error handling

## Formula 
BMI = Weight (kg )/ (Height (m) ** 2)

## BMI Classification
| BMI Range  | Category      |
|------------|---------------|
| Below 18.5 | Underweight   |
| 18.5-24.9  | Healthy       |
| 25-29.9    | Overweigth    |
| 30-34.9    | Class 1 Obese |
| 35-39.9    | Class 2 Obese |
| 40 or above| Class 3 Obese |

## How to Run 
1. Install Python3
2. open Terminal or Console in this folder 
3. Run: python bmi_calculator.py
